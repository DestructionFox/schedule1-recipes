# Effects

When mixing ingredients, substances are endowed with certain effects. Also, when mixing effects, new ones appear.

## Sample Combination
- Calorie-Dense + Balding = Sneaky

## Mix Strategy
Some characters require specific effects (e.g., Mac prefers: Focused, Spicy, Long Faced), so tailoring products boosts success.
